# Python Calculator Project

## Overview
This project contains two versions of a simple calculator implemented in Python.  
It demonstrates basic Python concepts such as user input, conditional statements, error handling, and type conversion.  

- **mark1**: Basic calculator with two numbers and an operator input.  
- **mark2**: Improved calculator with input validation, float support, and better error handling.

---

## Features

### Mark 1
- Takes two numbers as input
- Accepts an operator (`+`, `-`, `*`, `/`)
- Performs the operation and displays the result
- Basic and straightforward implementation

### Mark 2
- Accepts a single expression with spaces (e.g., `2 + 2`)
- Validates the input format (must have 3 elements: number operator number)
- Handles invalid numbers using `try/except`
- Converts numbers to float for more precise calculations
- Displays user-friendly messages for errors or invalid operators
